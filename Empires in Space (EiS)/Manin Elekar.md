Manin soon developed Dissociative Identity Disorder (DID) due to the trauma he experienced (Around like 8) Manin's type of DID is called "Correlative Dissociative Identity Disorder" (CDID). Correlative Dissociative Identity Disorder or CDID is a mental disorder where a person has multiple personalities that can interact with the main personality as well with other personalites. CDID is different from normal DID (both Possessive and Non-possessive DID). CDID causes the person to have multiple personalities that appear in the physical world but they are just mental beings so they can't interact with the world, they are kinda like hallucinations or very vivid imaginary beings. The amount of how many personalities created from CDID depends on the individual. More personalities can be created or destroyed over time, for example: a person started with 2 personalities, personality X and personality Y, currently the person has 5 personalities but lost personality X. This is not something the main personality can control as this is the making of the brain. These personalities can appear and disappear in/from the physical world whenever they want. The main personality cannot make a personality disappear but they can mute a personality, this only mutes the personality for the main personality and not the other personalities. The main personality can make a personality appear when they want. Personalities created from CDID can also interact with eachother, they aren't excluded to interact with only the main personality. Personalities created by CDID can control the body of their host if the main personality gives them permission to do so. When a personality controls the body, the main personality and other personalities get teleported inside the main personality's mental space. The personalities can't access the physical world when a personality created from CDID controls a body, they can only access the physical world when the main personality controls the body. The main personality also can't access the physical world when a personality takes over their body. The personality controlling the body can return it to the main personality or give it to another personality. The main personality can take over the body whenever they want to, so this forces personalities to negotiate with the main personality on how long they will have control over the body and what they are and aren't allowed to do. A personality can break rules set by the main personality when they control their body since the main or other personalities can't monitor them but this might break trust. There is a way to force a personality out of a body by putting them unconscious, when the person wakes up, it automatically sets it to the main personality. Personalites created from CDID appearances can be slightly or very different from the main personality's appearance, the difference usually lies in the uniform/outfit. The appearance of personalities can change if the appearance of the main personality (mainly the body of the main personality) changes or for other reasons. CDID develops in a person due to trauma, whether that be physical, mental, psychological, emotional trauma, like normal DID (in Manin's case, all of the above). It is a response from the brain to dull the trauma or guide the person from the trauma by creating multiple personalities. CDID can also take in some of the trauma inflicted so it dulls it for the main personality and therefore, slightly less painful, although, the personalities don't take all of it, they only take some of it. The personalities can also take in emotions aswell, mostly, negative ones so the main personality can stay calm enough. Each personality can take a different types of trauma. The main personality can experience the full trauma if it decides it wants to feel it, for some reason. CDID is called "Correlative Dissociative Identity Disorder" because each personality correlates with the person's emotions, ideology or something, like someone from the past or a character from a video game.
