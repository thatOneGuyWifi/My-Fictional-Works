I'd decided I'll have my own rule on how to use punctuation mark. I made this document to clearly layout the rules, use it as reference and update the rules as I go.

---

**Comma (,)**: Use it when two or more clauses (independent of type) are related in a sub-topic, in lists or list-like sentences (the last item in a list should end with "and", "or" or any other variant without a comma), when directly addressing someone or use when in need of a short pause.

**Colon (:)**: Use it when two or more clauses interact with eachother (one affecting the other) directly, following a "cause and effect" rule or when listing characteristics/properties of a subject/object.

**Semicolon (;)**: Use it when adding a related supplementary or optional sentence to another sentence or in sub-lists (a list inside a list: A,B,C;D,E,F;G,H,I 3 sub-list inside one list separated by two semicolons).

**Period (.)**: Use it when ending a sub-topic (but the main topic continues) or main topic.

**Ellipsis (...)**: Use it when in need of a long pause (that doesn't end the sub-topic) or when leaving out information.

**Em-dash (—)**: Use it like a parenthesis which has more important information than a regular parenthesis (do not add space to the outside words touching the em-dash) or when a clause is interrupted.

**Hyphen (-)**: Use it in when joining two words together to make one word (a combined word).

**Parenthesis (())**: Use it when adding optional information to words, clauses, sentences, paragraphs or anywhere in a topic.

**Square Brackets ([])**: Use it when adding optional information inside a parenthesis.

**Curly Brackets ({})**: Use it when adding optional information inside a square bracket.

**Slash (/)**: Use it as a "or"; when writing alternatives.

**Double Quotes ("")**: Use it when first announcing a new term, quoting a person/outside quotes, when turning a word that is not a noun into a noun or when a word or sentence isn't entirely true/accurate.

**Single Quotes ('')**: Use it in double quotes, they work the same as double quotes.

**Apostrophe (')**: Use it when contracting a word (combing two words into a smaller word [e.g: don't = do not]), when showing possession (it should be followed by an s) or forming plurals for letters, number and symbols.

**Question Mark (?)**: Use it after a sentence that is any type of question.

**Exclamation Mark (!)**: Use it when a sentence has emotional intensity—the exception being sadness and any sub-types of it—(When a sentence doesn't have a clear emotion to them and an exclamation mark is still used then it means excitement).

**Interrobang (?!/!?)**: Use it when a sentence is a type of question with emotional intensity; essentially a combination of the Question Mark and Exclamation Mark (it usually describes shock/surprise).

**Tilde (~)**: Use it when a word(s) is an approximation or in dialogue/monologue to indicate a sing-song/playful/teasing/flirty tone.

**Asterisk (\*)**: Use it before a word(s), clause or sentence to imply that there are important caveats to that word, clause or sentence.

> [!Note]
> Sub-topics can be defined loosely (in parameters) but generally they are topics inside a main topic
